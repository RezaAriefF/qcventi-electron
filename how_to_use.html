How To Use

1. Make sure the tool is in a good and correct position so that it does not affect how the tool works.
2. Connect the QC Ventilator to an energy source first.
3. Open the QC Ventilator Application on the desktop.
4. Turn on the QC Ventilator that has been turned on. Connect the desktop App to the QC Ventilator tool with a type-A serial cable.
5. Setting the application to connect to the QC Ventilator tool.
6. And the tool can be used according to the menus in the application.
7. To see the indicator can be seen in the flow and pressure sensor menu.
8. Maximize one of the sensor screens, it will display a more detailed view.
9. Minimize again to return to the summary of all sensors.
10. Setting the limit on the pressure sensor to determine the work of the pressure sensor then press the start and stop buttons to start and end the operation.
11. Then select the menu menu in the side bar to move to other applications.
12. To disable the tool, you can immediately shutdown on the logout menu and the tool will automatically disconnect itself.
